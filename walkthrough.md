# The Aspira Complete Redesign

We have successfully rebuilt the Diamond Estates landing page to entirely mirror the aesthetic and flow of `https://theaspira.vn/`.

## Key Changes
1. **Design System:** Shifted from deep blue branding to the signature Terracotta orange (`#ef6522`), and updated layout radiuses for pill-shaped buttons.
2. **Global Navigation:** Injected a global `<FloatingNav />` that features a left-side scroll-spy tracker (`01`-`05`) and right-side social action FABs (360°, Facebook, Zalo).
3. **Linear Flow:** Removed all generic agency-style sections (`GlobalMarkets`, `WhyUs`, `Testimonials`) to turn the landing page into a singular immersive project focus.
4. **Footer Overhaul:** Replaced standard footers with the massive `AspiraContact` component, showcasing a split solid-orange and glassmorphism interface.
